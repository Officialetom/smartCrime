Smart Crime Prediction Using Federated Learning Model for Nigeria Police
=======================================================================
1. Upload multiple CSVs (e.g., station1.csv, station2.csv) to train the global model.
2. Each CSV must contain columns: location, time, day, previous_crime, label.
3. After training, go to the Predict page.
4. Upload a prediction CSV (e.g., predict_data.csv) with columns: location, time, day, previous_crime.
5. Select time of day manually, and get prediction results with risk levels.
